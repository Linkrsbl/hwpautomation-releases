# HWP(한글) 로컬 제어 브릿지 명령어 가이드

이 문서는 ChatGPT, Claude 등 터미널(CLI) 조작이 가능한 AI 에이전트가 사용자의 로컬 환경에 설치된 HWP(한글) 데스크탑 앱을 원격제어하기 위한 명령어 가이드입니다. 

에이전트는 이 문서의 규칙을 완벽하게 숙지하고, 사용자가 요구하는 문서를 자동 작성해야 합니다.

---

## 🚀 브릿지 제어 기본 원칙

너는 `HwpBridge.exe`라는 외부 C# 콘솔 프로그램을 호출하여 현재 켜져 있는 한글 문서를 조종할 수 있다. 프로그램의 절대 경로는 사용자가 첫 프롬프트에서 알려줄 것이다. 터미널은 기본적으로 **PowerShell**을 사용한다.

**명령어 호출 형태 (필수 구조)**:
`& "[사용자가 알려준 브릿지 절대 경로]" "HWP_SECURE_TOKEN_2026" --command [명령어] [추가옵션]`
*(주의: 항상 두 번째 인자로 보안 토큰인 `HWP_SECURE_TOKEN_2026`을 반드시 넣어야 프로그램이 작동한다.)*

---

## 🛠️ 지원되는 명령어 (Commands)

### 1. 단순 텍스트 찾아 바꾸기 (Replace)
문서 내에서 특정 단어나 문장을 찾아 일괄 변경할 때 사용한다.
```powershell
& "브릿지경로" "HWP_SECURE_TOKEN_2026" --command replace --target "찾을단어" --text "바꿀단어"
```

### 2. 구조화된 데이터 자동 삽입 (Insert) - 핵심 기능
표 안에 데이터를 채워 넣거나, 특정 양식(글자 크기, 굵기 등)을 유지하며 방대한 텍스트를 삽입할 때 사용한다. 이 명령어는 JSON 파일을 인자로 받는다.

#### [실행 절차]
1. 삽입할 데이터를 분석하여 아래 예시와 같은 JSON 포맷 데이터 구조를 만든다.
2. 해당 JSON 데이터를 사용자의 PC 임시 경로(예: `C:\temp\payload.json`)에 파일로 저장한다.
3. 아래 명령어를 실행하여 브릿지가 파일을 읽고 문서에 주입하도록 명령한다.

```powershell
& "브릿지경로" "HWP_SECURE_TOKEN_2026" --command insert --file "C:\temp\payload.json"
```

#### [JSON 페이로드 구조 예시]
```json
{
  "projectName": "사업계획서 작성",
  "templateId": "템플릿 1",
  "sections": [
    { 
      "text": "시장 분석 결과 요약", 
      "attributes": { "bold": true, "size": 14 } 
    },
    { 
      "text": "1. 본 사업의 시장 규모는 지속적으로 성장하고 있으며...", 
      "attributes": { "bold": false, "size": 11 } 
    }
  ]
}
```

---

## 🎯 에이전트 행동 지침

1. 사용자가 문서 초안이나 텍스트를 제공하면, 기존 문서의 양식과 문맥을 파악해라.
2. 위에서 설명한 `insert` 명령어와 JSON 페이로드를 활용하여 한글 문서에 맞게 문단을 구성해라.
3. 터미널 명령어를 실행하여 최종적으로 HWP 문서에 텍스트가 모두 삽입되도록 완수해라.
